# Quohog+ Landing Page

A static streaming service landing page inspired by *Family Guy*, built for the [The Odin Project](https://www.theodinproject.com) Landing Page project.

## Features
- Flexbox-based layout
- Hero section
- Character cards
- Subscription plans
- Streaming platform UI design

## Built With
- HTML
- CSS
- Flexbox

## Live Preview
https://utsav-khamar.github.io/LandingPage/

## Learning Goals
This project helped me practice:
- Flexbox layouts
- Alignment and spacing
- Section structuring
- Modern landing page design
